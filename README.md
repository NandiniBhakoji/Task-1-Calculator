# Task 1 – Java Calculator

## About the Project

This project is a simple menu-driven calculator developed using Java. The program allows the user to perform basic arithmetic operations such as addition, subtraction, multiplication, and division by selecting options from a menu.

## Features

* Addition of two numbers
* Subtraction of two numbers
* Multiplication of two numbers
* Division of two numbers
* Division by zero handling
* User-friendly menu that runs until the user chooses to exit

## Technologies Used

* Java
* Scanner class for user input
* Methods and functions
* Loops and switch-case statements

## How the Program Works

1. The program displays a menu with different arithmetic operations.
2. The user selects an option and enters two numbers.
3. The corresponding operation is performed and the result is displayed.
4. The menu appears again until the user selects the Exit option.
